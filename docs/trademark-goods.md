# 한줄(hanjul) 상표 지정상품/서비스 명칭 초안 — 변리사 전달용

> 사업 정체성: **ebook 셀프퍼블리싱 + 책 서평단 매칭 + 글로벌 K콘텐츠 유통 플랫폼(웹/앱)**
> 기준: 특허청 「상품의 명칭과 류 구분 고시」 고시명칭 + 니스(NICE) 유사군코드.
> ⚠️ **모든 명칭은 KIPRIS 고시명칭 DB로 1건씩 최종 대조 전 초안**(고시명칭은 매년 개정, 비고시명칭은 보정요구 대상). 유사군 S/G코드 확정은 선행조사 단계에서 변리사가.
> 조사일: 2026-07-01

## 어떤 류를 낼 것인가

| 우선순위 | 류 | 왜 | 매핑 |
|---|---|---|---|
| **필수** | **41** | 전자출판·콘텐츠 제공·교육(서비스) | ebook 열람 제공, K콘텐츠 유통 |
| **필수** | **42** | 플랫폼·저작툴 SW를 서비스로(SaaS) | GitHub식 저작툴, 웹/앱 |
| **필수** | **35** | 온라인 마켓플레이스·판매대행·**중개(서평단 매칭)**·광고 | 셀프퍼블 판매·정산, 서평단 매칭 |
| 권장 | **9** | 다운로드형 ebook 파일·앱(상품) | 다운로드 전자책, 앱 |
| 선택 | **16** | 종이책·인쇄물(POD 확장 시) | 인쇄본 |

브랜드 방어 핵심 3류 = **41 + 42 + 35**. 예산 여유 시 9류, 종이책 계획 시 16류. **한글 문자상표 + 영문(hanjul) 문자상표 별도 출원** 권장.

## 41류 — 출판·콘텐츠 제공·교육
전자출판업 / 전자탁상출판업 / 서적출판업 / 잡지출판업 / 온라인 전자서적·잡지출판업 / **온라인 전자출판물 제공업(읽기전용)** / 온라인 전자서적 제공업(읽기전용) / 온라인 도서·전자서적 대여업 / 디지털 콘텐츠 제공업(온라인) / 오디오북 제공업·출판업 / 교육정보 제공업 / 온라인 교육정보 제공업 / 온라인 교육강좌업 / 교재 개발·편집업 / 오락·문화행사 기획업(북콘서트) / 웹툰·만화 콘텐츠 제공업(웹툰 로드맵) / 번역업(글로벌 다리)
> 함정: "온라인 전자출판물 제공업"은 흔히 **(읽기전용)** 한정 — 다운로드 아닌 열람. 이 형태여야 9류와 충돌 없음.

## 42류 — SaaS·플랫폼·SW 개발
서비스형 소프트웨어(SaaS)업 / SaaS 제공업 / 서비스형 플랫폼(PaaS) 제공업 / 컴퓨터 소프트웨어 플랫폼 제공업 / 애플리케이션 서비스 제공업(ASP) / 컴퓨터 소프트웨어 설계·개발업 / 응용솔루션 개발업 / **전자출판용** 컴퓨터 소프트웨어 개발업 / 모바일 애플리케이션 개발업 / 웹사이트 기획·개발·유지관리업 / 클라우드 컴퓨팅 서비스업 / 데이터 호스팅·웹호스팅업 / **온라인 비다운로드 소프트웨어 제공업(전자출판용)** / SW 유지·보수업 / 인공지능(AI) 소프트웨어 개발업(표지·웹툰 AI) / 온라인 보안서비스업
> 함정: 2021~ KIPO는 "컴퓨터 소프트웨어" 단독이면 **용도 명시 보정** 요구 → "전자출판용/저작·편집용" 필수. "비다운로드 SW 온라인 제공"이 SaaS 정통 표현(9류 다운로드형과 구분).

## 35류 — 쇼핑몰·판매대행·중개·광고
인터넷 종합쇼핑몰업 / 전기통신에 의한 통신판매 중개업 / 통신판매 대행업 / 온라인 마켓플레이스 운영업 / 상품 판매 대행·알선업 / **전자책 판매대행업** / 서적 도·소매업·전자출판물 소매업 / 상품·서비스 중개업(온라인) / 사업정보 제공업 / 온라인·인터넷 광고업 / 광고 대행업 / 광고물 기획·제작·배포업 / 마케팅·마케팅 대행업 / 판촉업 / 소비자 상품정보 제공업 / 시장·소비자 조사업(리뷰 데이터) / **상품·서비스 이용후기(리뷰) 제공업**
> 함정 & **서평단 핵심 판단**: 매칭/알선/**중개는 35류가 정석**. 서평·리뷰 **콘텐츠 제작·게시** 측면은 41류로도 → **35·41 크로스 지정** 권장. 판매·중개 명칭은 대상 상품(전자책·도서) 특정 필요.

## 9류 — 다운로드형 SW·전자출판물(상품)
내려받기 가능한: 전자출판물 / 전자서적(전자책) / 전자잡지·정기간행물 / 오디오북 / 컴퓨터 소프트웨어 / **전자출판용** SW(다운로드형) / 모바일 애플리케이션(전자출판용) / 디지털 콘텐츠(전자책·만화) · 전자책 단말기용 소프트웨어
> 함정(최대 실무이슈): **다운로드 ebook=9류(상품) ≠ 열람·스트리밍 제공=41류(서비스)**. 유사군 달라 자동 상호커버 안 됨. 한줄은 41류 본진 + 다운로드 제공 시 9류 병행.

## 16류 — 종이책·인쇄물(선택)
서적·도서 / 잡지·정기간행물 / 소책자·팸플릿 / 카탈로그·브로슈어 / 인쇄물 / 달력·포스터 / 문구류(북마크·스티커) / 그림책·만화책

## 함정 요약 (변리사 킥오프 체크)
1. **ebook 9류(다운로드·상품) vs 41류(열람·서비스)** — 유사군 달라 자동커버 안 됨.
2. **서평단 매칭 = 35류(중개) 정석**, 리뷰 콘텐츠 제작은 41류 → 크로스 지정.
3. **플랫폼/저작툴 자체 = 42류** — 35(판매)·41(제공)과 별개. 41·42·35는 겹침이 아니라 상호보완.
4. **SW 명칭 용도 특정 필수**(2021~ KIPO) — "전자출판용" 등.
5. **고시명칭 우선**(심사 원활·전자출원료 절감) — 최종 KIPRIS DB 대조.
6. **출원 후 류·지정상품 추가 불가**(요지변경) → 처음에 넉넉히(41·42·35 최소).
7. **글로벌**이면 향후 마드리드 국제출원(WIPO) 대비해 니스 국제표현과 정렬.
