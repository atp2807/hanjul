# 한줄 법률문서 필수 기재항목 체크리스트 (2025~2026)

> 한줄 = 자기명의 판매 **통신판매업자** + 작가 업로드 **OSP(호스팅)**. 판매물=다운로드형 전자책(디지털콘텐츠) → **청약철회 파트가 가장 예민.**
> ⚠️ 초안은 내가 → 전문가 검수 전제. 조문번호·서식은 law.go.kr 원문 최종대조.

## ★ 최중요: 전자책 청약철회 예외 (전자상거래법 §17②5호·§17⑥, 시행령 §21의2)
- 원칙: 계약서면 받은 날부터 **7일 내** 철회 가능.
- **디지털콘텐츠 예외**: "제공이 **개시**된 경우" 철회 불가(다운로드형 전자책). 단 **가분적**(회차·챕터)이면 **미제공 부분은 철회 가능**.
- **★ 예외를 쓰려면 반드시(§17⑥) — 안 하면 예외 무효 = 전액 환불해야 함**:
  1. **"청약철회 불가 사실"을 쉽게 알 수 있는 곳에 명확히 표시**, +
  2. **시험사용 조치**(시행령 §21의2): ①미리보기/미리듣기 ②한시적 체험 ③기능제한 체험본 ④(곤란시) 콘텐츠 정보 제공.
- **한줄 구현**: 결제 직전 "열람/다운로드 개시 시 철회 제한" **동의 체크박스** + **무료 미리보기 제공**(→ 코드에 `preview_block_cnt` 이미 있음, 결제 전 동의 게이트가 없음) + **제공 개시 시점 로그** 기록. **미성년자 취소권은 이 제한을 뚫음**(아래).

## 1. 이용약관 (약관규제법 §3·§6~14, 전자상거래법 §13②)
- [ ] 명시·설명 의무(§3): 미설명 조항은 계약내용 주장 불가.
- [ ] 변경 사전공지: 7일 전(**불리한 변경 30일 전**+유예).
- [ ] 필수 조항: 목적·정의 / 효력·변경 / 가입·**탈퇴** / 제공·중단 / 청약·성립 / 결제 / **청약철회·환불(디지털 제한)** / **저작권 귀속(★작가 보유 + 플랫폼은 판매·유통 라이선스 — 인세정산 구조 명문화)** / 손해배상·면책 / 분쟁·관할.
- [ ] 무효 조항 피하기(§7~14): 고의·중과실 면책, 과중 손배예정, 법정 해지권 배제, 급부 일방변경, 부당 관할·입증책임 전가.

## 2. 개인정보 처리방침 (개인정보보호법 §30①, 시행령 §31) — 초기화면 상시 게재
- [ ] ①처리 목적 ②항목 ③보유·이용기간 ④제3자 제공 ⑤**처리위탁**(PG·novelpotato·SFTP배포·클라우드 전부) ⑥파기 절차·방법 ⑦정보주체·법정대리인 권리·행사방법 ⑧**개인정보 보호책임자(CPO)** 연락처 ⑨쿠키 등 자동수집·거부 ⑩권익침해 구제(분쟁조정위·KISA 118).
- [ ] 시행령 §31 추가: 안전성 확보조치 / **국외이전**(해외 클라우드·글로벌 배포 시★) / 가명정보 / **자동화 결정(§37의2, 추천·랭킹 시)**.
- [ ] CPO 지정(§31): 초기엔 대표/책임자 지정으로 충분(전문CPO는 연매출 1,500억+대규모만). 방침엔 반드시 기재.

**코드로 요구되는 정보주체 권리 API**
- [ ] 열람(§35)·정정/삭제(§36)·처리정지(§37)·동의철회(§15·§22).
- [ ] **회원탈퇴+파기** — 단 **법정 보존항목 분리보관**(전자상거래법 §6: 계약·청약철회·대금결제·재화공급 기록 **5년**, 소비자불만·분쟁 3년). 인세정산 거래기록은 못 지우고 분리보존.
- [ ] **14세 미만**(§22의2): 수집 시 법정대리인 동의 필수 → **안 받을 거면 가입 연령차단이 안전.**
- [ ] ⚠️ **개인정보 이동권(전송요구권 §35의2) 정정**: 2025 시행됐지만 **의료·통신·에너지 등 지정분야 한정 → 이북 플랫폼은 아직 강제 아님.** 방침에 무리하게 약속 말 것(약속하면 이행책임 생김). GDPR 대비 글로벌 확장 시 별도.

## 3. 사업자정보 표시 (전자상거래법 §10·§12·§13①, 시행규칙 §4) — 초기화면
- [ ] 상호·대표자 / 사업장 주소 / 전화·이메일 / 사업자등록번호 / **통신판매업 신고번호+신고기관** / 이용약관 링크 / **호스팅제공자 상호(AWS·Cloudflare)**.
- [ ] **통신판매업 신고**: 면제=거래 50회 미만 **또는** 간이과세자. → **한줄은 유료판매로 곧 50회 초과 → 신고 필수.**

## 4. 청약철회·환불 정책 (전자상거래법 §17·§18)
- [ ] 일반 철회 7일 / **표시·계약 상이 시 3개월 내·안 날 30일 내**(디지털 제한과 무관 항상 인정).
- [ ] 환급 기한(반환일 3영업일 내, 지연배상금).
- [ ] 철회 제한 사유 전체(§17② 1~5호) + **각 사유마다 §17⑥ 사전표시+조치**(전자책=미리보기) 명시.

## 5. 청소년 보호정책 (정보통신망법 §42의3, 시행령 §25)
- [ ] 청소년보호책임자 지정 = (일평균 10만↑ **또는** 매출 10억↑) **&&** 청소년유해매체물 제공 시. → **성인물 안 팔면 지정의무 원칙 없음**(정책 페이지는 게시 권장).
- [ ] 19금 판매 시: 유해매체물 표시·**성인인증·접근차단**·책임자 공개·신고채널 필수.

## 6. 저작권 침해 신고·게시중단 (저작권법 §102·§103·§103의3) — 한줄=OSP
- [ ] **수신인(접수담당) 지정·공지**(§103④): 부서·이메일·연락처 공개 — **없으면 책임제한 못 받음.**
- [ ] 신고 접수(§103①: 중단요청서+소명) → **즉시 중단+권리자·업로더 양방 통보**(§103②).
- [ ] **재개 절차**(§103③): 업로더 소명 재개요청 → 권리자가 재개예정일 전 소제기 통보 없으면 재개. *(세부 기간·서식은 시행령 원문 확인)*
- [ ] 근거·벌칙(§136, 침해 5년↓/5천만↓, 친고죄) 고지. → 코드 신고큐 이미 있음, **정책문서화** 필요.

## 미성년자 결제·환불 (민법 §5·§6·§17·§141) — 약관·환불정책에 반영
- [ ] **취소권(§5)**: 만19세 미만 무단결제는 본인/법정대리인 **취소→전액환불(§141).** ★**이 취소권이 디지털 청약철회 제한(§17)보다 상위 — 다운로드했어도 취소 가능.**
- [ ] 예외: ①**속임수로 성년/동의 가장**(§17, 입증책임=사업자) ②용돈 범위(§6) ③권리만 얻는 행위.
- [ ] 구현: 가입 생년월일→미성년 판별, 결제 시 법정대리인 동의(본인인증)+기록 보존, 환불정책 명문화.
